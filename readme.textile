h1. Instructions

This is a sample rails app for serving event data in JSON format.

To download the code, type the following git command in terminal:

bc. git clone git@github.com:neilang/EventsFeed.git

This will copy the code from the repository to your computer.

To run the application, @cd@ into the downloaded directory and type:

bc. rake db:migrate
rails s

This will serve the application at @http://localhost:3000/@
